Weather App

"I developed a dynamic weather website that provides real-time weather forecasts upon user demand. Utilizing HTML, CSS, JavaScript, and Bootstrap framework, the website offers a simple and intuitive interface where users can search for weather forecasts by inputting the name of the desired city. The website fetches live weather data through an API and displays essential information such as temperature, humidity, wind speed, and additional details like sunrise and sunset times. By integrating Bootstrap for responsive design, the website ensures compatibility across various devices. This project showcases my proficiency in front-end web development, API integration, and user-centered design."

Skills

HTML
HTML5 is the latest version of Hypertext Markup Language, the standard language for creating web pages and applications. It introduces new features like semantic elements, audio and video support, canvas for drawing graphics, and improved accessibility, enabling developers to build more dynamic and interactive web experiences.
CSS
CSS3, or Cascading Style Sheets Level 3, is the latest evolution of CSS, a style sheet language used for describing the presentation of a document written in HTML. CSS3 introduces advanced features like transitions, animations, gradients, and responsive design techniques, allowing developers to style web pages with enhanced creativity and flexibility.
JavaScript
JavaScript is a high-level, dynamic programming language primarily used for front-end web development. It enables interactive and dynamic functionality on web pages, such as handling user interactions, manipulating the DOM (Document Object Model), and making asynchronous requests to servers. JavaScript is also widely used in back-end development (Node.js) and for building mobile and desktop applications.
C++
C++ is a powerful and versatile programming language widely used for system/application software, game development, and high-performance computing. Known for its efficiency and performance, C++ supports both procedural and object-oriented programming paradigms, offering features like strong typing, low-level memory manipulation, and rich standard libraries. C++ is commonly used in industries where performance and resource management are critical considerations.
